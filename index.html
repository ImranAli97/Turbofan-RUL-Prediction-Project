Turbofan RUL Prediction

End to end pipeline for remaining useful life estimation on the NASA C MAPSS dataset using PyTorch.

Readme Requirements Entry script Training Model Dataset Optimizer Utils

Overview

This project predicts remaining useful life for turbofan engines with a transformer style model and a clear training pipeline. It covers data loading, feature preparation, model training, and evaluation in a structure that is easy to reuse.

Features

  • Modular Python code for data, model, training, and utilities
  • Configurable training loop with metric tracking
  • Ready for experiments and logging

Project files

  • main.py entry point
  • training.py trainer logic
  • model.py network
  • dataset.py loaders
  • optimizer.py training config
  • utils.py helpers
  • README.md usage notes
  • requirements.txt packages

How to run locally

  1. Install Python and create a virtual environment
  2. Activate the environment
  3. Install the packages from the requirements file
  4. Place the dataset as described in the readme file
  5. Run the entry script
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
python -m venv .venv
# On Windows: .venv\Scripts\activate
# On macOS or Linux: source .venv/bin/activate
pip install -r requirements.txt
python main.py
