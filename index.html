Turbofan RUL Prediction

End to end pipeline for remaining useful life estimation on the NASA C MAPSS dataset using PyTorch.

Readme Requirements Entry script Training Model Dataset Optimizer Utils

Overview

This project predicts remaining useful life for turbofan engines with a transformer style model and a simple training pipeline. It includes data loading, feature prep, model training and evaluation in a clear structure that is easy to reuse.

Features

  • Clean Python modules for data, model, training, and utilities
  • Configurable training loop with metric tracking
  • Ready for experiments and result logging

Project files

  • main.py run entry
  • training.py trainer logic
  • model.py network
  • dataset.py loaders
  • optimizer.py train config
  • utils.py helpers
  • README.md usage notes
  • requirements.txt packages

How to run locally

  1. Install Python and a virtual environment tool
  2. Create and activate a virtual environment
  3. Install the packages listed in the requirements file
  4. Open the project in your editor
  5. Run the entry script to train or evaluate

Full steps are in the readme file linked above.
